Название: Получение списка авторов через API (позитивный тест)

Номер: A-001

Шаги:
1. Открыть Postman.
2. Выбрать метод запроса: Get.
3. В поле URL ввести: http://localhost:8080/authors
4. Нажать на кнопку "Send" для отправки запроса.

Ожидаемый результат:
Статус код ответа 200 Ok
В теле ответа возвращает список всех авторов в системе {json}.
