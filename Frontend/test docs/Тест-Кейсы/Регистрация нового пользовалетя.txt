Название:
Проверка регистрации пользователя (ввод невалидных данных в поле Password)

Шаги:
1. Перейти на сайт Alefowl.com
2. Нажать кнопку "Login" в правом верхнем углу
3. Нажать кнопку Don't have an account? Sign Up
3. Заполнить поля "Login", "First Name", "Surname", "Email", "Password"
   (в поле Password ввести не валидные данные)
4. Нажать кнопку "Register"

Ожидаемый результат:
Пользователь не зарегестрирован на сайте, появляется сообщение об ошибке
